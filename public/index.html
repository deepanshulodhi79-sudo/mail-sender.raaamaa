💡 App Password Guide

Gmail ya Outlook ke liye normal password kaam nahi karta.
Gmail App Password:
1. Google Account → Security → 2-Step Verification ON karein
2. myaccount.google.com/apppasswords par jaayein
3. "Mail Pro" naam se password banayein (16 characters milenge)

Outlook App Password:
1. Microsoft Account → Security → Advanced Security Options
2. App Passwords section mein banayein

⚡ Rate Limiting

Spam se bachne ke liye max 10 mails per minute ki limit hai.
Queue feature use karein bulk mails ke liye — automatic throttling hota hai.
